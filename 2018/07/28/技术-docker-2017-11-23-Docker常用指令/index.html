Docker 常用指令

作为一个开发最近才开始使用docker ，深感惭愧！
最近在学习 CloudMan 大佬的 《每天5分钟玩转 Docker 容器技术》, 通俗易懂，每天积累一点点，积累，这里记录一下常用的 docker 相关知识，仅做笔记用。

  1. 查看本地的所有 docker镜像

    docker images

  2. 查看指定镜像信息

    docker images ubuntu

  3. 查看指定tag 的镜像信息

    docker images ubuntu:latest

  1. 从 Docker Hub 下载镜像

    docker pull hello-world

  2. 交互运行一个 container

    docker run -it ubuntu
